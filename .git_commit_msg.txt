OPT-15: SQ8 raw distance for graph navigation (+6.6% QPS, +0.44pp recall)

- Add l2_sq8_raw_avx2: skip scale_sq weighting, accumulate as i32
  Eliminates 24 instructions per distance computation (SIFT-128)
  2x cvtepi32_ps + 2x loadu_ps + 2x fmadd_ps -> 2x add_epi32 per 32-dim chunk
- Add const RAW: bool generic to greedy_search_sq8 (zero branch, zero duplication)
- Make raw the default search_sq8 path (search_sq8_weighted for backward compat)
- A/B/A/B experiment (5 rounds, 100K queries/round, graph cache):
  Weighted: mean QPS=14,907, recall=0.9636
  Raw:      mean QPS=15,900, recall=0.9680
  Delta: +6.6% QPS, +0.44pp recall (win-win)
- Root cause: scale_sq weighting amplifies SQ8 quantization error on high-variance
  dimensions, distorting distance ranking. Raw distance better preserves true
  nearest-neighbor relationships for graph navigation.
